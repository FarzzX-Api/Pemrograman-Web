Contoh Penggunaan Web

Berikut adalah beberapa contoh penggunaan teknologi web seperti CSS, JavaScript, Python, PHP, C++, dan C#.

1. CSS (Cascading Style Sheets)

CSS digunakan untuk menata elemen-elemen HTML.

/* Contoh penggunaan CSS */
body {
    background-color: lightblue;
}
h1 {
    color: navy;
    font-size: 2em;
}
p {
    color: darkgray;
}

2. JavaScript

JavaScript digunakan untuk membuat halaman web interaktif.

Ini adalah teks yang akan berubah.

3. Python (Server-Side)

Python biasanya digunakan di sisi server, berikut adalah contoh sederhana dengan Flask:

# app.py (Flask)
from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Hello, Python Flask!"

if __name__ == "__main__":
    app.run()

4. PHP (Server-Side)

PHP digunakan untuk menangani logika di sisi server.

<?php
    echo "Hello, World from PHP!";
?>

5. C++ (Ilustrasi Konsep)

C++ biasanya digunakan untuk pemrograman sistem atau aplikasi desktop. Berikut contoh sederhana C++ yang tidak berjalan di web browser:

// contoh.cpp
#include <iostream>
using namespace std;

int main() {
    cout << "Hello, World from C++!" << endl;
    return 0;
}

6. C# (ASP.NET Core)

C# digunakan untuk membuat aplikasi web dengan framework ASP.NET Core. Berikut adalah contoh controller sederhana di C#:

// HomeController.cs
using Microsoft.AspNetCore.Mvc;

namespace WebApp.Controllers
{
    public class HomeController : Controller
    {
        public IActionResult Index()
        {
            return View();
        }

        public IActionResult About()
        {
            ViewData["Message"] = "Halaman tentang aplikasi ini.";
            return View();
        }
    }
}

Untuk menjalankan aplikasi ini, pastikan Anda memiliki lingkungan ASP.NET Core dan .NET SDK yang sesuai.
